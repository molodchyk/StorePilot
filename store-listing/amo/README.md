# AMO Store Listing

This folder contains StorePilot's Firefox Add-ons listing copy and media references.

- `listing/en-US.md` contains the English (US) AMO listing fields.
- `media/screenshots.md` records the screenshot upload order, captions, and source artifact paths.

## AMO Listing Text Rules

AMO listing text is not the same contract as Chrome Web Store listing text.

- The summary is limited to 250 characters.
- The description can be longer, but should stay scan-friendly and user-facing.
- Some AMO listing fields support a limited set of Markdown.
- The description can use bullets, links, bold, italic, blockquotes, and fenced code.
- AMO uses one shared screenshot set, although screenshot descriptions can be localized.

Use `docs/amo-submission.md` for reviewer notes, release notes, validation notes, source-package answers, and other submission workflow details.
